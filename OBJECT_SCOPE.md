# Object Scope

## 环境说明
当前处于开发环境（DEV）。
原则上允许修改开发环境中的对象，但所有可运输对象的变更必须绑定到 Transport Request。

## 允许 AI 操作的对象

详见 [ABAP_ENV.md](ABAP_ENV.md) 中的允许访问范围。

## 约束

- 修改前优先确认对象所属包、传输层和影响范围
- 如果对象属于标准对象或跨系统共享对象，修改前必须先确认
- 如果涉及激活、发布、删除、数据模型变更或权限变更，必须先确认

## Transport Request

- 每次修改都必须挂接到 Transport Request
- 如果当前没有请求，先创建请求，再执行修改
- 修改完成后，说明请求号、修改内容和影响范围
